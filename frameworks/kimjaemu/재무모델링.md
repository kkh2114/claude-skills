# 재무 모델링 프레임워크

> 현장/회사 수익성 분석, 시나리오 시뮬레이션, 원가 관리 체계.
> 참고: harness-100 #53 financial-modeler 기반, 건설/인테리어 특화.

## 1. 현장 수익성 분석 모델

### 기본 구조
```
매출(계약금액) - 직접공사비 - 간접공사비 - 일반관리비 = 영업이익
영업이익 / 매출 × 100 = 영업이익률
```

### 직접공사비 분류
| 항목 | 내용 | 관리 포인트 |
|------|------|-----------|
| 재료비 | 자재 구매비 | 시세 변동, 할인 협상 |
| 노무비 | 작업자 인건비 | 단가×투입일수, 성수기 할증 |
| 외주비 | 하도급 비용 | 계약 대비 실행 관리 |
| 경비 | 가설, 폐기물, 장비 | 예산 대비 집행률 |

### 원가율 관리
```
원가율 = 실제 집행액 / 계약금액 × 100

정상: 65~80%
주의: 80~90%
위험: 90% 이상 (적자 전환 우려)
```

## 2. 시나리오 분석

### 3시나리오 모델
| 시나리오 | 가정 | 적용 |
|---------|------|------|
| **낙관 (Bull)** | 설계변경 없음, 공기 단축, 자재비 하락 | 최선 |
| **기본 (Base)** | 현재 계획 그대로 | 기준 |
| **비관 (Bear)** | 설계변경 추가, 공기 지연, 자재비 상승 | 최악 |

### 변수별 민감도
| 변수 | ±10% 변동 시 영업이익 영향 |
|------|-------------------------|
| 계약금액 | 직접 ±10% |
| 재료비 | 약 ±4~5% |
| 노무비 | 약 ±3~4% |
| 공사기간 | 간접비 증가 (1개월당 약 3~5%) |

## 3. 현금흐름 관리

### 자금 흐름표
```
월초 잔액
+ 기성 수금
+ 전도금 수령
- 재료비 지출
- 노무비 지출
- 외주비 지출
- 경비 지출
- 일반관리비
= 월말 잔액
```

### 자금 경색 조기 경보
| 지표 | 정상 | 주의 | 위험 |
|------|:----:|:----:|:----:|
| 현금 잔고 | 1개월치 지출 이상 | 2주치 | 1주치 미만 |
| 기성 미수금 | 30일 이내 | 30~60일 | 60일 초과 |
| 전도금 소진율 | 공정률과 동기화 | 공정률+10% | 공정률+20% |

## 4. DCF 간이 분석 (프로젝트 투자 판단용)

```
NPV = Σ(현금흐름t / (1+r)^t) - 초기투자

r = 할인율 (건설업 통상 8~12%)
IRR = NPV가 0이 되는 할인율
```

| 판단 | 기준 |
|------|------|
| 투자 적합 | IRR > 할인율, NPV > 0 |
| 검토 필요 | IRR ≈ 할인율 |
| 투자 부적합 | IRR < 할인율, NPV < 0 |
