# 가격 전략 프레임워크

> 견적 가격 체계, 원가 분석, 수익성 시뮬레이션.
> 참고: harness-100 #50 pricing-strategy 기반, 건설/인테리어 특화.

## 1. 견적 가격 산정 구조

```
직접공사비 (재료+노무+외주+경비)
+ 간접공사비 (현장관리비, 가설비)
+ 일반관리비 (본사비용 배부)
+ 이윤
= 도급금액 (VAT 별도)
```

### 이윤율 기준
| 공사 유형 | 통상 이윤율 | 비고 |
|----------|:---------:|------|
| 관급 공사 | 15% 이내 | 예정가격 기준 |
| 민간 소규모 | 20~35% | 협상 여지 |
| 인테리어 리모델링 | 25~40% | 디자인 부가가치 포함 |
| 설계시공 일괄 | 30~45% | 설계비 포함 |

| 방법 | 절감 가능 비율 | 리스크 |
|------|:------------:|--------|

| 잔재 활용 | 1~3% | 미미 |

## 4. 견적 시뮬레이션

```
[시뮬레이션]
기본 견적: 250,000,000원 (이윤율 30%)

시나리오 A: 이윤율 25% → 230,769,000원 (▼8%)
시나리오 B: 이윤율 35% → 269,231,000원 (▲8%)
시나리오 C: 자재비 10% 절감 → 242,500,000원 (▼3%)
```
