# 세금 계산 프레임워크

> 법인세, 부가세, 원천세 관리 및 절세 전략.
> 참고: harness-100 #75 tax-calculator 기반, 건설업 법인 특화.

## 1. 건설업 주요 세금

| 세금 | 과세 기준 | 신고 시기 |
|------|----------|----------|
| 법인세 | 사업연도 소득 | 사업연도 종료 후 3개월 |
| 부가가치세 | 매출세액 - 매입세액 | 분기별 (1/25, 4/25, 7/25, 10/25) |
| 원천세 | 급여, 사업소득, 기타소득 | 익월 10일 |
| 지방소득세 | 법인세의 10% | 법인세와 동시 |

## 2. 부가세 관리 (건설업 특화)

### 기성 청구 시 부가세
```
기성액: 92,800,000원
부가세: 9,280,000원
청구 총액: 102,080,000원
전도금 상계: 34,000,000원
수령액: 68,080,000원
```

### 매입세액 공제 관리
| 항목 | 공제 가능 | 주의사항 |
|------|:--------:|---------|
| 자재 구매 (세금계산서) | ✅ | 적격 세금계산서 필수 |
| 노무비 (일용직) | ❌ | 면세 — 공제 불가 |
| 외주비 (하도급) | ✅ | 세금계산서 수취 확인 |
| 경비 (유류, 통행료) | ✅ | 사업용 카드/세금계산서 |
| 접대비 | 제한 | 1만원 초과 시 적격증빙 |

### 부가세 예상액 계산
```
매출세액 (기성 청구) - 매입세액 (자재+외주+경비) = 납부세액
분기별로 추적하여 자금 계획에 반영
```

## 3. 법인세 절세 포인트 (건설업)

| 항목 | 절세 방법 | 효과 |
|------|----------|------|
| 감가상각 | 정률법 선택 (초기 비용 인정 극대화) | 법인세 이연 |
| 퇴직급여충당금 | 한도 내 설정 | 비용 인정 |
| 대손충당금 | 매출채권 대비 한도 설정 | 비용 인정 |
| 연구개발비 | R&D 세액공제 (해당 시) | 세액 직접 감면 |
| 중소기업 특별세액감면 | 건설업 20% 감면 | 법인세 20% 감면 |
| 고용증대 세액공제 | 신규 채용 시 | 1인당 700~1,100만원 |
