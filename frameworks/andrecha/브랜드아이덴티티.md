# 브랜드 아이덴티티 프레임워크

> 상업공간의 브랜드 아이덴티티를 공간 디자인에 반영하는 체계.
> 참고: harness-100 #06 brand-identity 기반, 공간디자인 연계 특화.

## 1. 브랜드 → 공간 변환 프로세스

```
브랜드 분석 → 공간 언어 도출 → 디자인 가이드라인 → 적용 → 검증
```

## 2. 브랜드 분석 체크리스트

| 항목 | 질문 | 공간 반영 |
|------|------|----------|
| 브랜드 미션 | "왜 존재하는가?" | 공간의 존재 이유 |
| 타깃 고객 | "누구를 위한 공간인가?" | 동선, 좌석, 분위기 |
| 브랜드 성격 | "사람이라면 어떤 성격?" | 소재, 컬러, 조명 |
| 핵심 가치 | "무엇을 가장 중시?" | 디자인 우선순위 |
| 경쟁 포지션 | "경쟁자와 뭐가 다른가?" | 차별화 포인트 |

## 3. 브랜드 성격 → 공간 언어 변환표

| 브랜드 성격 | 공간 소재 | 컬러 | 조명 | 가구 |
|-----------|----------|------|------|------|
| 친근한 | 원목, 패브릭 | 웜톤 | 따뜻한 간접조명 | 라운드형, 쿠션 |
| 전문적 | 콘크리트, 메탈 | 모노톤 | 밝고 균일 | 직선형, 심플 |
| 고급스러운 | 대리석, 브라스 | 다크+골드 | 포인트 조명 | 클래식, 중후 |
| 활기찬 | 컬러풀 타일, 네온 | 비비드 | 밝고 다이내믹 | 캐주얼, 스툴 |
| 자연적 | 석재, 식물, 나무 | 어스톤 | 자연광 극대화 | 오가닉 형태 |

## 4. 공간 톤앤매너 가이드라인 양식

```
[공간 톤앤매너 가이드라인]
══════════════════════════════
■ 브랜드명:
■ 공간 키워드: (3~5개)
■ 컬러 시스템: 베이스 / 서브 / 악센트
■ 소재 팔레트: 메인 / 서브 / 포인트
■ 조명 방향: (전체조명 / 간접조명 / 포인트조명 비율)
■ 가구 방향: (스타일, 소재, 형태)
■ DO: (해야 할 것)
■ DON'T: (하면 안 되는 것)
```

## 5. 사이니지/그래픽 연계

| 요소 | 공간 적용 | 주의사항 |
|------|----------|---------|
| 로고 | 파사드, 카운터 뒤 | 크기/위치 가이드라인 준수 |
| 서체 | 메뉴판, 안내문 | 브랜드 서체 일관 적용 |
| 컬러 | 인테리어 악센트 | 브랜드 컬러와 동일 |
| 패턴 | 벽지, 타일 | 과하지 않게 포인트만 |

## 6. 산출물 양식

```
[브랜드-공간 연계 보고서]
══════════════════════════════
1. 브랜드 분석 요약
2. 브랜드 성격 → 공간 언어 변환
3. 톤앤매너 가이드라인
4. 적용 예시 (무드보드)
5. 김전무 보고용 핵심 요약
```
