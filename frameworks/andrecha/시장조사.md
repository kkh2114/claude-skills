# 시장 조사 프레임워크

> 인테리어/공간디자인 시장 분석, 트렌드, 경쟁 분석 체계.
> 참고: harness-100 #44 market-research 기반, 공간디자인 특화.

## 1. 시장 조사 프로세스

```
산업 분석 → 경쟁 분석 → 소비자/트렌드 분석 → 기회 도출 → 보고
```

## 2. 인테리어 산업 분석 프레임워크

### 시장 규모 분석 (TAM-SAM-SOM)
| 구분 | 정의 | 인테리어 적용 |
|------|------|-------------|
| **TAM** | 전체 시장 | 국내 인테리어 시장 전체 |
| **SAM** | 접근 가능 시장 | 상업공간 인테리어 (서울/수도권) |
| **SOM** | 확보 가능 시장 | 카페/식음 공간 리모델링 |

### 산업 동향 체크리스트
- [ ] 시장 성장률 (전년 대비)
- [ ] 주요 성장 드라이버
- [ ] 규제 변화 (건축법, 소방법 등)
- [ ] 기술 변화 (스마트 건축, 친환경 등)
- [ ] 원자재 가격 동향

## 3. 경쟁 분석 (Porter's 5 Forces)

| 요인 | 분석 질문 | 인테리어 적용 |
|------|----------|-------------|
| 기존 경쟁자 | 주요 경쟁사 수, 차별화 수준 | 지역 내 인테리어 업체 |
| 신규 진입 위협 | 진입 장벽 높이 | 저가 업체 진입 용이 |
| 대체재 위협 | 다른 방법으로 해결 가능? | 셀프 인테리어, 온라인 플랫폼 |
| 공급자 교섭력 | 자재/인력 공급 상황 | 자재비 변동, 인력 수급 |
| 구매자 교섭력 | 고객의 선택권 | 비교 견적 용이, 가격 민감도 |

## 5. 기회 도출 (SWOT)

| | 긍정 | 부정 |
|---|------|------|
| **내부** | 강점 (Strengths) | 약점 (Weaknesses) |
| **외부** | 기회 (Opportunities) | 위협 (Threats) |

### SO 전략 (강점+기회)
→ 공격적 확장 영역

### WT 전략 (약점+위협)
→ 방어/보완 영역
