# 공간 컨셉보드 프레임워크

> 스타일 분석, 무드보드, 컬러 팔레트, 아이템 리스트 체계.
> 참고: harness-100 #77 space-concept-board 기반.

## 1. 컨셉보드 작성 프로세스

```
스타일 분석 → 무드보드 설계 → 컬러 팔레트 → 자재/가구 큐레이션 → 예산표 → 프레젠테이션
```

## 2. 스타일 분석 체계

### 인테리어 스타일 분류
| 스타일 | 키워드 | 대표 소재 | 컬러 톤 |
|--------|--------|----------|---------|
| 미니멀 모던 | 심플, 깔끔, 기능 | 콘크리트, 철, 유리 | 화이트, 그레이, 블랙 |
| 내추럴 | 따뜻함, 자연, 편안 | 원목, 린넨, 식물 | 베이지, 그린, 브라운 |
| 인더스트리얼 | 거친, 빈티지, 노출 | 벽돌, 철, 시멘트 | 그레이, 브라운, 블랙 |
| 스칸디나비안 | 밝음, 기능, 아늑 | 자작나무, 패브릭 | 화이트, 파스텔, 우드 |
| 재패니즈 모던 | 절제, 여백, 조화 | 목재, 종이, 돌 | 베이지, 차콜, 내추럴 |
| 뉴트로 | 복고, 레트로, 감성 | 테라조, 아치, 타일 | 오렌지, 민트, 크림 |
| 럭셔리 | 고급, 우아, 클래식 | 대리석, 브라스, 벨벳 | 골드, 네이비, 다크그린 |

### 상업공간 용도별 가이드
| 용도 | 핵심 고려사항 | 추천 스타일 방향 |
|------|-------------|---------------|
| 카페 | 체류시간, 사진촬영, 동선 | 내추럴, 인더스트리얼, 뉴트로 |
| 레스토랑 | 좌석 효율, 조명, 프라이버시 | 럭셔리, 재패니즈, 미니멀 |
| 오피스 | 집중/협업 균형, 브랜딩 | 미니멀 모던, 스칸디 |
| 리테일 | 동선, 진열, 브랜드 경험 | 브랜드 종속 (커스텀) |
| 병원/클리닉 | 청결감, 안정감, 기능 | 미니멀, 스칸디 |

## 3. 무드보드 구성 요소

| 요소 | 내용 | 수량 |
|------|------|:----:|
| 레퍼런스 이미지 | 유사 공간 사례 | 5~8장 |
| 컬러 팔레트 | 메인/서브/악센트 | 3~5색 |
| 자재 샘플 | 바닥, 벽, 천장 마감 | 3~5종 |
| 가구 이미지 | 핵심 가구 | 3~5점 |
| 조명 이미지 | 조명 기구 | 2~3점 |
| 텍스처 | 패브릭, 소재 질감 | 2~3종 |
| 키워드 | 공간 분위기 형용사 | 3~5개 |

## 4. 컬러 팔레트 설계

### 배색 비율 원칙 (60-30-10)
| 역할 | 비율 | 적용 | 예시 |
|------|:----:|------|------|
| **베이스** | 60% | 벽, 천장, 대면적 | 화이트, 라이트그레이 |
| **서브** | 30% | 바닥, 가구, 커튼 | 우드톤, 베이지 |
| **악센트** | 10% | 소품, 조명, 포인트 | 골드, 딥그린 |

### 컬러 팔레트 양식
```
[프로젝트명] 컬러 팔레트
─────────────────────────
■ 베이스: #FFFFFF (화이트) — 벽면, 천장
■ 서브1:  #D4C5A9 (웜베이지) — 바닥
■ 서브2:  #8B7355 (월넛) — 가구
■ 악센트: #2D5016 (딥포레스트) — 식물, 소품
■ 포인트: #C8A951 (앤틱골드) — 조명, 금속
```
