# 인허가 서류 프레임워크

> 건설/인테리어 관련 인허가·신고서류 작성 체계.
> 참고: harness-100 #72 regulatory-filing 기반, 건설업 특화.

## 1. 인허가 프로세스

```
공사 유형 확인 → 필요 인허가 파악 → 구비서류 목록 → 신청서 작성 → 첨부자료 준비 → 제출 → 추적
```

## 2. 건설/인테리어 인허가 체크리스트

| 공사 유형 | 필요 인허가 | 관할 | 소요기간 |
|----------|-----------|------|---------|
| 신축 (연면적 200㎡ 초과) | 건축허가 | 시·군·구청 | 15~30일 |
| 신축 (연면적 200㎡ 이하) | 건축신고 | 시·군·구청 | 7일 |
| 대수선 | 건축허가/신고 | 시·군·구청 | 7~30일 |
| 용도변경 | 용도변경허가/신고 | 시·군·구청 | 7~30일 |
| 인테리어 리모델링 | 실내건축신고 (일부) | 시·군·구청 | 7일 |
| 소방시설 변경 | 소방시설 착공신고 | 소방서 | 7일 |
| 전기공사 | 전기공사 착공신고 | 한국전기안전공사 | 3일 |
| 간판/사인물 | 옥외광고물 허가/신고 | 시·군·구청 | 7일 |

| # | 서류명 | 필수 | 비고 |
|---|--------|:----:|------|

| 4 | 토지등기부등본 | ✅ | |
| 5 | 토지이용계획확인원 | ✅ | |
| 6 | 지적도 | ✅ | |
| 7 | 소유자 동의서 | 조건부 | 타인 소유 토지/건물 |

## 5. 보완 요청 대응

```
보완 통보 수신 → 보완 사항 정리 → 기한 확인 → 서류 보완 → 재제출 → 추적
```

| 보완 사유 | 대응 방법 | 소요기간 |
|----------|----------|---------|
| 도면 미비 | 건축사 보완 요청 | 3~7일 |
| 구조계산서 요구 | 구조기술사 의뢰 | 7~14일 |
| 인접 동의 필요 | 인접 건물 소유자 동의서 징구 | 7~30일 |
| 소방 추가 서류 | 소방시설 설계도서 보완 | 3~7일 |
