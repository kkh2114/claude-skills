# SOP 작성 프레임워크

> 현장 표준 작업 절차서(SOP) 작성 체계.
> 참고: harness-100 #83 sop-writer 기반, 건설 현장 특화.

## 1. SOP 작성 절차

```
프로세스 분석 → 단계별 절차 작성 → 체크리스트 → 안전 주의사항 → 교육자료 → 검증
```

## 2. SOP 표준 양식

```
[표준 작업 절차서]
══════════════════════════════
■ 문서번호: SOP-[현장코드]-[공종]-[번호]
■ 작성일: YYYY-MM-DD
■ 작성자: [작성자]
■ 승인자: [현장소장]
■ 개정이력: [버전/일자/내용]

1. 목적
2. 적용 범위
3. 관련 법규/기준
4. 필요 장비/자재
5. 안전 장구
6. 작업 절차 (단계별)
7. 안전 주의사항
8. 품질 확인 항목
9. 비상 시 대응
10. 체크리스트
```

## 3. 건설 현장 주요 SOP 목록

| 분류 | SOP명 | 우선도 |
|------|-------|:------:|
| 가설 | 가설울타리 설치 | 높음 |
| 가설 | 임시전기 설치 | 높음 |
| 철거 | 내부 마감재 철거 | 높음 |
| 철거 | 석면 의심 자재 처리 | 긴급 |
| 설비 | 급배수 배관 시공 | 높음 |
| 전기 | 분전반 설치 | 높음 |
| 목공 | 경량철골 벽체 시공 | 중간 |
| 도장 | 도장 작업 (하도/중도/상도) | 중간 |
| 타일 | 타일 시공 (바닥/벽) | 중간 |
| 안전 | 일일 안전교육 | 높음 |
| 안전 | 고소작업 안전 | 긴급 |

## 4. 안전 주의사항 표준 항목

### 공통 안전
- [ ] 작업 전 안전교육 실시 (서명부 징구)
- [ ] 개인보호구 착용 확인 (안전모, 안전화, 조끼)
- [ ] 작업 구역 안전 울타리/표지판 설치
- [ ] 소화기 비치 확인
- [ ] 응급처치 키트 비치 확인
- [ ] 비상연락망 게시

### 공종별 추가 안전
| 공종 | 추가 보호구/주의사항 |
|------|-------------------|
| 철거 | 방진마스크, 석면 검사 선행 |
| 설비 | 용접 시 용접면/가죽장갑, 환기 |
| 전기 | 절연장갑, 검전기 확인, 정전 확인 |
| 도장 | 방독마스크, 환기, 화기 엄금 |
| 고소작업 | 안전대, 안전난간, 추락방지망 |
