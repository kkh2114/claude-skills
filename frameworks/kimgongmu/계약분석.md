# 계약 분석 프레임워크

> 건설/인테리어 계약서 검토 시 적용하는 분석 체계.
> 참고: harness-100 #66 contract-analyzer 기반, 건설업 특화 재구성.

## 1. 분석 절차

```
계약서 수신 → 구조 파악 → 조항별 분석 → 리스크 등급 → 수정안 → 보고
```

## 2. 조항별 분석 방법 (IRAC)

| 단계 | 설명 | 예시 |
|------|------|------|
| **I**ssue | 이 조항에서 문제될 수 있는 쟁점 | "하자보수 기간이 과도한가?" |
| **R**ule | 관련 법률·판례·업계 관행 | "민법 제671조, 건설업 표준 1~3년" |
| **A**pplication | 이 계약에 적용하면 | "5년 하자보수는 업계 표준 대비 과다" |
| **C**onclusion | 결론 및 권고 | "3년으로 수정 협상 권고" |

## 3. 리스크 등급 분류

| 등급 | 점수 | 의미 | 조치 |
|------|------|------|------|
| 🔴 **HIGH** | 80~100 | 즉시 시정 필요, 계약 거부 검토 | 대표님 즉시 보고 |
| 🟡 **MEDIUM** | 40~79 | 협상 필요, 수정안 제시 | 김전무 보고 후 판단 |
| 🟢 **LOW** | 0~39 | 수용 가능, 참고 사항 | 기록만 |

## 4. 건설/인테리어 계약 필수 점검 항목

| 필수 조항 | 미비 시 리스크 | 업계 표준 |
|----------|--------------|----------|
| 공사 범위 | 추가공사 분쟁 | 설계도서 기준 명시 |
| 공사 기간 | 지연 책임 분쟁 | 착공~준공일 명시 |
| 계약 금액 | 기성 분쟁 | 총액/단가 계약 구분 |
| 기성 지급 조건 | 자금 경색 | 월 기성, 검수 후 30일 이내 |
| 설계변경 절차 | 추가비용 미수 | 서면 합의 + 금액 조정 |
| 하자보수 기간 | 과도한 의무 | 1~3년 (공종별 차등) |
| 지체상금률 | 과다 손실 | 1일 계약금액의 0.1% |
| 해지/해제 조건 | 일방적 해지 위험 | 중대 위반 시 시정 최고 후 해지 |
| 손해배상 한도 | 무제한 배상 위험 | 계약금액 한도 |
| 분쟁 해결 | 불리한 관할 | 공사현장 소재지 관할 |

## 5. 위험 조항 패턴 (건설 특화)

### 🔴 치명적 위험
| 패턴 | 대표 문구 | 수정 방향 |
|------|----------|----------|
| 무제한 손해배상 | "일체의 손해를 배상" | 계약금액 한도 설정 |
| 일방적 해지권 | "갑은 이유 불문하고 해지" | 중대 위반 + 시정 최고 후 해지 |
| 지체상금 과다 | "1일 계약금액의 1%" | 0.05~0.1% (업계 표준) |
| 하자보수 과다 | "10년간 하자보수" | 공종별 1~3년 |
| 기성 지급 지연 | "준공 후 60일" | 월 기성, 검수 후 30일 |

### 🟡 중간 위험
| 패턴 | 대표 문구 | 수정 방향 |
|------|----------|----------|
| 설계변경 비용 미반영 | "변경 시 추가비용 없음" | 서면 합의 + 정산 조항 |
| 검수 기간 무제한 | "갑이 만족할 때까지" | 14일 이내, 미통보 시 간주검수 |
| 전도금 한도 초과 | 한도 명시 없음 | 계약금액의 20% 한도 명시 |

## 6. 협상 전략 (건설업)

### 대금 지급 개선안
| 현재 조건 | 개선안 | 교환 제안 |
|----------|--------|----------|
| 준공 후 일시불 | 월별 기성 지급 | "기성 시 이행보증보험 제출" |
| 검수 후 60일 | 검수 후 30일 | "간주검수 조항 추가" |
| 전도금 없음 | 착공 시 20% 선급 | "선급이행보증보험 제출" |

### 리스크 항목 수정 화법
```
"업계 표준 관행과 관련 판례에 비추어, 
 [구체적 조항]은 시공사에게 과도한 부담이 됩니다.
 [수정안]으로 조정하면 양측 리스크를 균형 있게 관리할 수 있습니다."
```

## 7. 산출물 양식

```
[계약 검토 보고서]
══════════════════════════════
1. 계약 개요 (유형, 당사자, 금액, 기간)
2. 필수 조항 체크리스트 (✅/❌)
3. 리스크 매트릭스 (등급별 분류)
4. 🔴 HIGH 항목 상세 분석 (IRAC)
5. 수정 권고안
6. 협상 전략
7. 김전무 보고용 핵심 요약
```
