# 구매 문서 프레임워크

> 자재/설비 구매 시 벤더 비교, 평가, 검수까지의 체계.
> 참고: harness-100 #95 procurement-docs 기반, 건설 자재 구매 특화.

## 1. 구매 프로세스

```
요구사항 정의 → 벤더 조사 → 비교 평가 → 계약 조건 검토 → 발주 → 검수
```

## 2. 벤더 평가 스코어카드

| 카테고리 | 가중치 | 세부 항목 |
|---------|:------:|----------|
| 가격 | 30% | 단가, 운송비 포함 총비용, 할인 조건 |
| 품질 | 30% | 자재 등급, 인증(KS 등), 불량률 |
| 납기 | 20% | 납기 준수율, 긴급 대응 가능 여부 |
| 신뢰도 | 20% | 기존 거래 실적, 재무 안정성, A/S |

### 점수 산정
```
총점 = (가격점수×0.3) + (품질점수×0.3) + (납기점수×0.2) + (신뢰도점수×0.2)
각 항목 100점 만점
```

## 4. 검수 기준서 양식 (건설 자재)

### 자재 검수 체크리스트
| 검수 항목 | 기준 | 방법 | 합격 | 불합격 |
|----------|------|------|:----:|:------:|
| 수량 확인 | 발주서 대비 100% | 전수 계수 | | |
| 규격 확인 | 발주 규격 일치 | 실측/서류 확인 | | |
| 외관 검사 | 파손/변형 없음 | 육안 검사 | | |
| 인증 확인 | KS/ISO 인증서 | 서류 확인 | | |
| 시험성적서 | 요구 성능 충족 | 서류 확인 | | |

### 판정 기준
| 판정 | 조건 | 조치 |
|------|------|------|
| 합격 | 전 항목 합격 | 입고 처리, 대금 지급 |
| 조건부 합격 | 경미한 하자 | 교체분 재납품 후 정산 |
| 불합격 | 규격 미달/대량 파손 | 반품, 재발주 |

## 5. 건설 자재 특화 점검

### 주요 자재별 검수 포인트
| 자재 | 핵심 검수 항목 |
|------|--------------|
| 배관 (PVC/STS) | 규격(관경), 두께, KS인증, 피팅 호환 |
| 경량철골 | 규격(스터드/런너), 도금 상태, 직진성 |
| 석고보드 | 두께(9.5T/12.5T), 파손 여부, 방수/방화 등급 |
| 타일 | 규격, 색상 로트 일치, 파손률, 흡수율 |
| 전선 | 규격(sq), HIV/FR-CV 구분, KS인증, 길이 |
| 조명기구 | 모델 일치, 작동 테스트, 해외 납품 시 인증(KC) |
