# 전략 프레임워크

> 회사 전략 수립, OKR, SWOT, 의사결정 지원 체계.
> 참고: harness-100 #47 strategy-framework 기반.

## 1. OKR (Objectives & Key Results)

### OKR 설계 양식
```
[Objective] 정성적 목표 (영감을 주는, 도전적인)
  ├── [KR1] 정량적 핵심 결과 (측정 가능)
  ├── [KR2] 정량적 핵심 결과
  └── [KR3] 정량적 핵심 결과
```

### OKR 품질 체크
| 체크 | 기준 |
|:----:|------|
| ☐ | Objective가 영감을 주는가? (숫자가 아닌 방향) |
| ☐ | KR이 측정 가능한가? (숫자로 표현) |
| ☐ | KR 달성 시 Objective가 달성되는가? |
| ☐ | 70% 달성이 도전적 수준인가? (너무 쉽지 않은가) |
| ☐ | 분기 단위로 설정했는가? |

### 에스유디자인 OKR 예시
```
[O] 수주 경쟁력을 업계 상위 수준으로 끌어올린다
  [KR1] 분기 신규 수주 3건 이상
  [KR2] 제안서 채택률 40% 이상
  [KR3] 고객 재계약률 60% 이상
```

## 2. SWOT 분석

| | 긍정 | 부정 |
|---|------|------|
| **내부** | **S** 강점 | **W** 약점 |
| **외부** | **O** 기회 | **T** 위협 |

### 전략 도출 매트릭스
| 전략 | 조합 | 방향 |
|------|------|------|
| **SO** | 강점+기회 | 적극 공략 |
| **WO** | 약점+기회 | 약점 보완 후 기회 활용 |
| **ST** | 강점+위협 | 강점으로 위협 방어 |
| **WT** | 약점+위협 | 방어/철수/구조 변경 |

## 3. 의사결정 프레임워크

### RICE 우선순위 (프로젝트/안건 비교)
```
RICE 점수 = (Reach × Impact × Confidence) / Effort

Reach: 영향 범위 (1~10)
Impact: 효과 크기 (1~5)
Confidence: 확신도 (0.5~1.0)
Effort: 투입 노력 (인·주 단위)
```

### 의사결정 매트릭스 양식
| 안건 | 기준1 (가중치) | 기준2 (가중치) | 기준3 (가중치) | 총점 |
|------|:---:|:---:|:---:|:---:|
| 안A | /10 | /10 | /10 | |
| 안B | /10 | /10 | /10 | |
