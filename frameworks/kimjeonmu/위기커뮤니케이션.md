# 위기 커뮤니케이션 프레임워크

> 긴급 상황 시 메시지 전략, 대응 체계, 보도 대응.
> 참고: harness-100 #87 crisis-communication 기반.

## 1. 위기 대응 프로세스

```
상황 파악 → 등급 판단 → 메시지 수립 → 대내외 소통 → 모니터링 → 사후 복구
```

## 2. 위기 등급 분류

| 등급 | 기준 | 대응 수준 |
|------|------|----------|
| 🔴 Level 3 | 인명사고, 법적 소송, 언론 보도 | 대표님 직접 대응, 전문가 투입 |
| 🟡 Level 2 | 공기 대폭 지연, 발주처 클레임, 재정 위기 | 김전무 총괄, 대표님 보고 |
| 🟢 Level 1 | 경미한 품질 이슈, 민원, 내부 갈등 | 담당 팀원 대응, 김전무 보고 |

## 3. 위기 메시지 프레임워크 (4R)

| 단계 | 내용 | 핵심 |
|------|------|------|
| **R**egret | 유감 표명 | 진정성 있는 공감 |
| **R**eason | 원인 설명 | 사실 기반, 투명하게 |
| **R**emedy | 해결 방안 | 구체적 조치 + 일정 |
| **R**eform | 재발 방지 | 구조적 개선 약속 |

## 4. 건설 현장 주요 위기 시나리오

| 시나리오 | 등급 | 즉시 조치 | 소통 대상 |
|---------|:----:|----------|----------|
| 안전사고 (인명) | 🔴 | 119 신고, 현장 보존 | 발주처, 노동청, 유가족 |
| 공기 대폭 지연 | 🟡 | 원인 분석, 만회 계획 | 발주처, 하도급사 |
| 하자 클레임 | 🟡 | 현장 조사, 보수 계획 | 발주처, 시공팀 |
| 자금 경색 | 🟡 | 긴급 자금 확보 | 대표님, 거래처 |
| 주민 민원 | 🟢 | 소음/분진 저감 조치 | 주민, 구청 |

## 5. 산출물 양식

```
[위기 대응 보고서]
══════════════════════════════
1. 상황 요약 (5W1H)
2. 등급 판정
3. 즉시 조치 내역
4. 메시지 전략 (4R)
5. 소통 계획 (대상별)
6. 대표님 보고용 핵심 요약
```
