# 시나리오 기획 프레임워크

> 핵심 변수별 시나리오 설계, 영향 분석, 대응 전략 수립 체계.
> 참고: harness-100 #52 scenario-planner 기반.

## 1. 시나리오 기획 프로세스

```
핵심 변수 정의 → 시나리오 매트릭스 → 영향 분석 → 대응 전략 → 트리거 설정
```

## 2. 시나리오 매트릭스 (2×2)

| | 변수A: 긍정 | 변수A: 부정 |
|---|-----------|-----------|
| **변수B: 긍정** | 🟢 최선 | 🟡 혼합A |
| **변수B: 부정** | 🟡 혼합B | 🔴 최악 |

## 3. 시나리오별 상세 양식

```
[시나리오명]
■ 확률: %
■ 가정 조건:
■ 영향 (매출/비용/공기):
■ 대응 전략:
■ 트리거 신호: (이 신호가 나타나면 이 시나리오 진입)
■ 즉시 조치:
```

## 4. 건설업 핵심 변수

| 변수 | 낙관 | 기본 | 비관 |
|------|------|------|------|
| 자재비 변동 | -5% | 0% | +15% |
| 공사기간 | -1개월 | 예정대로 | +2개월 |
| 추가공사 | 없음 | 소규모 | 대규모 설계변경 |
| 수금 속도 | 15일 | 30일 | 60일+ |
| 인력 수급 | 충분 | 보통 | 부족 |

## 5. 산출물 양식

```
[시나리오 분석 보고서]
══════════════════════════════
1. 핵심 변수 정의
2. 시나리오 매트릭스
3. 시나리오별 영향 분석
4. 대응 전략 (시나리오별)
5. 트리거 모니터링 계획
6. 대표님 보고용 핵심 요약
```
