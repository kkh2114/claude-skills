# CAD Drawing JSON Spec Format

## 전체 구조

```json
{
  "title_block": { ... },
  "walls": [ ... ],
  "doors": [ ... ],
  "windows": [ ... ],
  "rooms": [ ... ],
  "dimensions": [ ... ],
  "furniture": [ ... ]
}
```

## title_block

```json
{
  "enabled": true,
  "project": "홍길동 아파트 리모델링",
  "title": "평면도",
  "scale": "1:100",
  "date": "2026-03-25"
}
```

## walls

모든 좌표는 **mm 단위**. (0,0)은 좌하단 기준.

```json
{
  "x1": 5000, "y1": 5000,
  "x2": 15000, "y2": 5000,
  "thickness": 200,
  "layer": "WALL"
}
```

- `thickness`: 기본 200mm (외벽), 내벽은 100~150mm → `layer: "WALL-INT"`
- `layer`: "WALL" (외벽) 또는 "WALL-INT" (내벽)

## doors

```json
{
  "x": 8000, "y": 5000,
  "width": 900,
  "angle": 0,
  "swing": "left"
}
```

- `width`: 문 너비 mm (일반 900, 화장실 750, 현관 1000)
- `angle`: 벽 기준 회전각 (0=수평, 90=수직)
- `swing`: "left" 또는 "right"

## windows

```json
{
  "x1": 10000, "y1": 5000,
  "x2": 12000, "y2": 5000
}
```

벽 위에 위치하는 두 점. 방향은 벽과 동일.

## rooms

방 라벨 (방 중심점에 텍스트 배치)

```json
{
  "x": 10000, "y": 10000,
  "name": "거실",
  "area": 25.5
}
```

- `area`: ㎡ 단위 면적 (선택)

## dimensions

치수선

```json
{
  "x1": 5000, "y1": 5000,
  "x2": 15000, "y2": 5000,
  "offset": 800
}
```

- `offset`: 벽에서 치수선까지 거리 mm

## furniture

가구 (사각형으로 표현)

```json
{
  "x": 6000, "y": 6000,
  "width": 2000, "height": 1000,
  "name": "소파"
}
```

## 한국 아파트 일반 치수 참고

| 항목 | 치수 (mm) |
|------|-----------|
| 외벽 두께 | 200~300 |
| 내벽 두께 | 100~150 |
| 일반 문 | 900 × 2100 |
| 화장실 문 | 750 × 2100 |
| 현관문 | 1000 × 2100 |
| 일반 창 | 1200~2400 |
| 발코니 창 | 2400~4000 |
| 1평 = 3.3㎡ | 약 1818 × 1818mm |

## 평수별 총 면적 참고

| 평수 | 전용면적 (㎡) | 대략 크기 (mm) |
|------|-------------|---------------|
| 20평 | 66 | 약 8100 × 8100 |
| 25평 | 82.5 | 약 9100 × 9100 |
| 30평 | 99 | 약 9900 × 9900 |
| 34평 | 112 | 약 10600 × 10600 |
| 40평 | 132 | 약 11500 × 11500 |
